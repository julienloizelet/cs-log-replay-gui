# CrowdSec Log Replay

Replay small log files through a local CrowdSec instance and view generated alerts with line-by-line explanations.

## What This Does

- Replay NGINX or Syslog log lines through CrowdSec
- View generated alerts and decisions
- See `cscli explain` output for each line
- Understand which scenarios match your logs

## Pre-installed

The environment comes with:
- **CrowdSec** with NGINX and LINUX collections
- The Log Replay GUI ready to use
